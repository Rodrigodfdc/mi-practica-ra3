¿Qué es GitHub Pages?

GitHub Pages es un servicio de hosting estático gratuito que permite publicar sitios web directamente desde un repositorio. No necesitas servidor ni configuración compleja.

💡 Nota: Solo admite sitios estáticos (HTML, CSS, JS). No ejecuta código en el servidor.

Pasos para publicar

1

Crear repositorio en GitHub

Repositorio público con un README inicial.

2

Añadir index.html

Crear el archivo de entrada con tu contenido HTML.

3

Activar GitHub Pages

Settings → Pages → rama main → / (root) → Save.

4

Acceder a tu URL

https://<usuario>.github.io/<repositorio>
